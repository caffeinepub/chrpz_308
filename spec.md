# Specification

## Summary
**Goal:** Update the app's color scheme to USA-themed colors (patriotic red, white, and navy blue) across the entire frontend.

**Planned changes:**
- Replace CSS custom properties in `index.css` with USA palette: patriotic red (`#B22234`), navy blue (`#3C3B6E`), and white (`#FFFFFF`)
- Update Tailwind config color tokens to match the new USA palette
- Apply the new colors consistently to backgrounds, text, buttons, headers, cards, accents, borders, and interactive elements
- Remove all remnants of the old dark reddish theme

**User-visible outcome:** The entire app displays a USA-themed red, white, and blue color scheme consistently across all UI elements.
